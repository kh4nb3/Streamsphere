# Streamsphere

A premium streaming platform prototype built with Django.

## Setup

1.  Install dependencies (if not already installed):
    ```bash
    pip install django
    ```

2.  Run the server:
    ```bash
    python manage.py runserver
    ```

3.  Open your browser and navigate to `http://127.0.0.1:8000/`.

## Features

-   **Home**: Trending movies and new releases with hover effects.
-   **Movie Detail**: Video placeholder, details, and comments.
-   **Subscription**: Pricing plans.
-   **Auth**: Login and Signup UI.
-   **Theme**: Premium Blue Dark theme.
